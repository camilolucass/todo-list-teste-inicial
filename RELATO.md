# Relato rapido

Organizei a entrega começando pelos requisitos obrigatorios: cadastro, listagem, edicao, exclusao, pesquisa, campos da tarefa e validacoes. Em seguida, defini uma solucao web simples, com foco em clareza de codigo e facilidade de publicacao.

Utilizei HTML, CSS e JavaScript puro, com `localStorage` para salvar as tarefas no navegador. Tambem utilizei IA como apoio no fluxo de desenvolvimento e revisao da implementacao.

Optei por nao incluir backend ou banco externo porque o escopo pode ser atendido com uma aplicacao estatica, mais simples de avaliar e publicar na Vercel. A interface foi separada em arquivos de HTML, CSS e JavaScript para manter a organizacao e facilitar manutencao futura.
